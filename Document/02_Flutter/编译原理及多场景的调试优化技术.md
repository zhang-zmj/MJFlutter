#### 一、Flutter编译方式：
1、编译：

```
1、Debug模式：打开了Assert，包含所有的调试信息，支持Hot reload
2、Release模式：关闭了所有Assert，包大小做了优化，禁止所有调试辅助
```

2、运行：

```
1、AOT：运行前编译，在程序运行前，已编译成对应平台的机器码，可以直接运行
2、JIT：即时编译，将代码编译成机器码，提高运行效率
```
